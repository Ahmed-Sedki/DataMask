# Data Mask

DataMask is a web-based tool that allows users to anonymize sensitive network information such as IP addresses, MAC addresses, and hostnames. The tool replaces sensitive data with mock information to ensure privacy.

## Features
- Anonymize IPv4 and IPv6 addresses
- Anonymize MAC addresses
- Anonymize hostnames
- Copy anonymized data to clipboard

## Usage

1. Clone the repository:
    ```bash
    git clone https://github.com/Ahmed-Sedki/DataMask.git
    ```
2. Navigate to the project directory:
    ```bash
    cd DataMask
    ```
3. Open `index.html` in your web browser to use the tool.
